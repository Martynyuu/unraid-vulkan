# Changelog

All notable changes to this project will be documented in this file.

## [2026.03.02] - 2026-03-02

### Added
- Initial release
- Vulkan Loader from Slackware packages
- Vulkan Tools (vulkaninfo)
- SPIR-V tools
- NVIDIA ICD configuration
- Docker container support with volume mounts
- Unraid Web UI status page
- Automatic NVIDIA driver detection
- Build script for package creation
- GitHub Actions release workflow
- Community Applications template

### Requirements
- Unraid 6.10.0 or later
- NVIDIA GPU with Vulkan support
- NVIDIA Driver plugin
